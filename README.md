# NetGrapher

NetGrapher is a small project to visualize how layer dimensions change while studying neural network architectures.

## Live demo

You can use it here: https://riccardotornesello.github.io/netgrapher/

## Why this project

I created NetGrapher from a personal need while learning neural networks.  
I am not an expert in this field, and AI tools helped me build this project, so there may still be mistakes.
